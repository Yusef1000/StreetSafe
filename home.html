Problem

In todays society, crime is on the rise and innocent civilians are facing unnecessary harm on a daily basis. According to many statistics, criminal activity has increased and has been especially prevalent here in New York City.

Solution



Safe Street


What's Safe Street?

In a nutshell, SafeStreet shows users which route is the safest for them to take.

How?

SafeStreet plots criminal activities on the map in a red color. The more red the certain area, the more you should avoid it.


But What If Im Already In The Red?

If you already find yourself in the red ares, please note the blue area on the map. These areas are known as "safe zones". They are public buildings to can escape to if in need.



How?



With The Power of Python!


We extract all of the data of crimes in NYC from csv files found on the NYC Open Data Website. In these files we only need the latitude and longitude of each crime. The coordinates are then plotted onto a map using Python's "basemap".
